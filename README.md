# Business Trip Reports — Group 5

A responsive, single-page academic portfolio containing five fictional business trip reports for **BUS 430 Section 4** at **Paragon International University**.

## Open the website

Open `index.html` in any modern web browser. No installation or local server is required.

## Edit before submission

- Update the date or any final course requirements in the Group 5 section near the end of `index.html`.
- Change report writing and figures directly in the matching report section.
- Images are loaded from Unsplash. Replace an image `src` with a local file if an offline version is needed.

## Project files

- `index.html` — report content and page structure
- `styles.css` — responsive visual design and print styles
- `script.js` — mobile menu, section highlighting, fade-ins, and back-to-top button
